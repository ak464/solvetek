# Project Roadmap

## Completed Milestones
- [x] **Core Foundation**: Next.js 16 setup, Tailwind 4, Supabase integration.
- [x] **Content Engine**: Tiptap editor, Article dynamic routes, Category pages.
- [x] **Visual Identity**: "SolveTek" branding, Arabic typography (Tajawal), Responsive design.
- [x] **Traffic Tools**: "Mobile Package Recommender" (removed per user request).
- [x] **Monetization V1**:
    - [x] Affiliate Link Manager.
    - [x] Global Ad Settings.
    - [x] Sidebar Product Recommendations.
- [x] **Dynamic Management**:
    - [x] Dynamic Header/Footer categories.
    - [x] Admin Category Manager (CRUD + Sorting).
- [x] **Advanced Articles Management** (2000+ Articles Support):
    - [x] Search & Filtering (text, category, status, author).
    - [x] Enhanced Pagination (20/50/100 per page).
    - [x] Bulk Actions (publish, unpublish, delete, category change).
    - [x] Statistics Dashboard (total, published, drafts, views).
    - [x] Professional Display (thumbnails, dates, author names).
    - [x] Performance Optimizations (indexes, skeletons, 90%+ improvement).

## Planned Features (Short Term)
- [ ] **Comment System**: Enable `comments` table usage on frontend.
- [ ] **Search v2**: Real-time search with debounce and search result snippets.
- [ ] **SEO Enhancements**: Auto-generate sitemap.xml dynamically (if not fully automated yet).

## Future Expansion (Long Term)
- [ ] **User Accounts**: Allow visitors to save articles (Bookmarks).
- [ ] **Interactive Diagnostics**: Step-by-step troubleshooting wizards for common issues (e.g., "Why is my internet slow?").
- [ ] **Mobile Application**: React Native app reusing Supabase backend.
- [ ] **AI Assistant**: RAG-based chatbot using article content to answer user questions directly.

## Maintenance
- **Regular**: Update NPM packages.
- **Monitoring**: Check Supabase usage quotas (Database size, Bandwidth).
